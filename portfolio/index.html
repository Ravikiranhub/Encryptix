This project focuses on creating a real-time sign language detection system. The project involves using TensorFlow for building and training CNN models, OpenCV for image processing and real-time capture, and NumPy and Scikit-Learn for data manipulation and analysis. Developed in PyCharm, it integrates machine learning applications in communication.
